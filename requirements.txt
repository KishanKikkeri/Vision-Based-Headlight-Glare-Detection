# ESP32-CAM Headlight Glare Detection System
# Requirements File

# =========================
# SOFTWARE REQUIREMENTS
# =========================

Arduino IDE (version 1.8.x or above OR Arduino IDE 2.x)

ESP32 Board Package (Espressif Systems)
URL: https://dl.espressif.com/dl/package_esp32_index.json

Required Arduino Libraries:
- esp_camera.h (included with ESP32 board package)

USB-to-Serial Drivers (for FTDI / programmer, if required)


# =========================
# HARDWARE REQUIREMENTS
# =========================

Core Components:
- ESP32-CAM (AI Thinker module recommended)
- FTDI Programmer / USB-to-Serial Module
- Jumper Wires

Optional Output Components:
- LED (for glare indication)
- Buzzer (for alert system)

Optional Components:
- Breadboard
- External Power Supply (5V recommended for stability)

Optional Enhancements:
- Relay Module (for automatic headlight dimming)
- OLED Display (I2C, e.g., SSD1306)
- WiFi-enabled monitoring device (phone/laptop)


# =========================
# SYSTEM REQUIREMENTS
# =========================

- Computer/Laptop
- USB Port
- Operating System:
  - Windows / macOS / Linux


# =========================
# NOTES
# =========================

- No external libraries required beyond ESP32 package
- esp_camera.h comes pre-installed with ESP32 board package
- Ensure stable 5V power supply for ESP32-CAM
- GPIO0 must be connected to GND during upload mode
- Use Arduino IDE Serial Monitor at 115200 baud rate
- Remove GPIO0 from GND after uploading code
